1. *Difference between var, let and const!*

"var" is old type variable. It can redeclare and also can change value many time.It can be console before declared, it can confuse us.  

"let" is mainly used as variable when value is not constant. It can change value but cannot redeclare in same block scope. 

"const" use for constant value. Once declare it cannot change again,but can push data in there.


2. *What is the spread operator (...)!*

- Spread operator `(...)` use to expand values from array or object.It used to combine or copy data.


3. *Difference between map(), filter(), and forEach()!*

"map()" enter in an array and return new array after doing some changes.  
"filter()" also return new array with all the elements that match condition.  
"forEach()" used to loop of the elements is an array
